# Task Manager

A Flutter application developed as part of a Flutter Developer Assessment.

## Features

- Login screen with email & password validation
- Fetch posts from REST API
- Search posts by title
- View post details
- Mark/Unmark favorite posts
- Favorites stored using SharedPreferences
- Pull-to-refresh support
- Loading indicator and error handling
- Responsive UI

## Tech Stack

- Flutter
- Provider (State Management)
- HTTP
- SharedPreferences

## Flutter Version

- Flutter 3.32.5

## Packages Used

- provider
- http
- shared_preferences
